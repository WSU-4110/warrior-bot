## Welcome!

warrior-bot is a CLI tool for Wayne State University staff and students. The idea is to make many of the things we need day to day easier to access and use through the terminal.

## Quick Start

Install warrior-bot:

```bash
pip install warrior-bot
```
>
> This will not work yet, as we have not published anything on the package index thus far.
> To install, clone the repo and run `pip install -e .`

Use the CLI:

"wb" is a shorthand alias for "warrior-bot"
```bash
# Find information
wb where naresh mahabir

# Navigate to resources
wb go degree works

# Booking rooms
wb book --b state hall
```
